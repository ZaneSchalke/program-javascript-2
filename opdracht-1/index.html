Gebruik van For en While Lussen

In deze opdracht beschrijven we wanneer je als developer een for of while lus gebruikt in je code. Hieronder volgen twee situaties voor elke lus.

For Loop

Een for loop wordt vaak gebruikt wanneer je van tevoren weet hoeveel iteraties er moeten worden uitgevoerd, of wanneer je een specifiek bereik van elementen wilt herhalen.

Situatie 1: Itereren over een lijst van producten

Stel, je hebt een lijst van producten in je webshop en je wilt door elk product in deze lijst itereren om ze weer te geven op je website. Je weet precies hoeveel producten er zijn en je wilt elke iteratie van de lijst doorlopen.


    const producten = ["Product 1", "Product 2", "Product 3"];
    
    for (let i = 0; i < producten.length; i++) {
        console.log(producten[i]);
    }

In dit voorbeeld gebruiken we een for loop omdat we weten dat de lijst van producten een vast aantal items heeft.

Situatie 2: Itereren over de getallen van 1 tot 10

Stel, je wilt de getallen van 1 tot 10 afdrukken. Omdat je precies weet hoeveel getallen er zijn (namelijk 10), kun je een for loop gebruiken om de herhalingen van 1 tot 10 te regelen.


    for (let i = 1; i <= 10; i++) {
        console.log(i);
    }

In dit voorbeeld gebruiken we een for loop omdat we het aantal herhalingen kennen.

While Loop

Een while loop wordt vaak gebruikt wanneer je niet precies weet hoeveel herhalingen er nodig zijn, maar wanneer een bepaalde voorwaarde nog steeds waar is. De loop blijft draaien zolang de voorwaarde waar is.

Situatie 1: Wacht totdat een gebruiker inlogt

Stel, je hebt een systeem waarbij de gebruiker moet inloggen. Je weet niet hoeveel tijd het duurt totdat de gebruiker zich aanmeldt, maar je wilt de code blijven controleren totdat de gebruiker inlogt. Dit is een typisch scenario voor een while loop.


    let ingelogd = false;
    
    while (!ingelogd) {
        // Controleer of de gebruiker is ingelogd
        ingelogd = checkInlogStatus();  // Stel dat deze functie true of false retourneert
    }
    console.log("Gebruiker is ingelogd!");

De loop blijft draaien totdat de gebruiker ingelogd is, wat een onbekende hoeveelheid tijd kan duren.

Situatie 2: Artikelen ophalen uit een database totdat er geen artikelen meer zijn

Stel, je haalt artikelen op uit een database en wilt blijven ophalen totdat er geen artikelen meer beschikbaar zijn. De exacte hoeveelheid artikelen is onbekend, maar je wilt doorgaan met ophalen totdat de database leeg is. Een while loop is hier geschikt.


    let artikelen = haalArtikelenOp();
    
    while (artikelen.length > 0) {
        console.log(artikelen);
        artikelen = haalArtikelenOp();  // Haal nieuwe artikelen op
    }

De while loop blijft doorgaan zolang er artikelen beschikbaar zijn om op te halen.
